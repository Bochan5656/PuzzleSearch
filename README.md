# PuzzleSearch
<p>2024年8月作成</p>
<p>パズル＆ドラゴンのスクリーンショットから盤面を認識し、経路探索を行うプログラム。</p>
<h2>ファイルの説明</h2>
<p>パズドラ解析ツール.py:メインのプログラム</p>
<p>ScreenShot:パズドラのスクショ保存先</p>
<p>β:メインのプログラムのバックアップ（作成時に使用）</p>

## 1. 環境構築手順

以下のコマンドを実行して、仮想環境の構築および必要なライブラリのインストールを行ってください。

```bash
# conda環境の作成とアクティベート
conda create -n PDroot python=3.9
conda activate PDroot

# 必要なライブラリのインストール
pip install opencv-python
```

## 2. このリポジトリの使い方

1. `ScreenShot` フォルダ内に、解析の基準となる各色の画像（`赤.PNG`, `緑.PNG`, `青.PNG`, `黄.PNG`, `紫.PNG`, `ピンク.PNG`）が配置されていることを確認します。
2. 解析したいパズドラの盤面スクリーンショット（例: `IMG_0028.PNG`）を `ScreenShot` フォルダ内に配置します（ファイル名はソースコードに合わせて変更してください）。
3. ターミナルで `パズドラ解析ツール.py` を実行します。
   ```bash
   python パズドラ解析ツール.py
   ```
4. ターミナルに以下が出力されます。
   - 認識した盤面の配列
   - スワイプを開始する座標 `(x, y)`
   - 最適なルート（`UP`, `DOWN`, `LEFT`, `RIGHT` の手順）
   - 最終的な予想コンボ数
